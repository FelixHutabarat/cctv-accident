Cara Kerja

Arsitektur CNN yang dibangun dari nol

Preprocessing Gambar

Gambar input di-resize ke 224×224px lalu dinormalisasi ke skala [0, 1] dengan membagi nilai piksel dengan 255. Dataset di-shuffle hanya pada data training untuk menjaga konsistensi evaluasi.

3 Blok Konvolusi

Tiga blok Conv2D + MaxPooling dengan filter 32→64→128 mengekstrak fitur visual bertahap — dari tepi dan tekstur di layer awal hingga pola kompleks kecelakaan di layer dalam. Total 22.2 juta parameter.

Visualisasi feature map Conv2D

Klasifikasi Biner

Flatten → Dense(256, ReLU) → Dense(1, Sigmoid). Output sigmoid menghasilkan probabilitas 0–1, dengan threshold 0.5 sebagai pemisah kelas Accident vs Non-Accident.
